ChapelWitch.coffee

¶

Gain one Chapel and one Witch, and otherwise play Big Money. One of the most powerful two-card strategies there is.

 
{
  name: 'ChapelWitch'
  gainPriority: (state) -> [
    "Colony" if state.current.countInDeck("Platinum") > 0
    "Province" if state.countInSupply("Colony") <= 6
    "Witch" if state.current.countInDeck("Witch") == 0
    "Duchy" if 0 < state.gainsToEndGame() <= 5
    "Estate" if 0 < state.gainsToEndGame() <= 2
    "Platinum"
    "Gold"
